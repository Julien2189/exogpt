JavaScript — Tableaux d'objets

Tri des tableaux d'objets avec sort()

Dans ce cours, tu vas revoir toutes les méthodes importantes pour trier des tableaux d'objets en JavaScript : nombres, textes, dates, filtres, calculs, copie de tableau et récupération du premier élément.

Tableau de départ

On va travailler avec un tableau d'objets. Chaque objet représente un produit avec plusieurs propriétés : nom, prix, stock et categorie. 

const produits = [
  { nom: "Clavier", prix: 49, stock: 12, categorie: "informatique" },
  { nom: "Souris", prix: 25, stock: 30, categorie: "informatique" },
  { nom: "Écran", prix: 199, stock: 5, categorie: "informatique" },
  { nom: "Chaise", prix: 89, stock: 8, categorie: "bureau" },
  { nom: "Bureau", prix: 249, stock: 3, categorie: "bureau" },
  { nom: "Lampe", prix: 35, stock: 15, categorie: "bureau" }
];
Attention : la méthode sort() modifie directement le tableau original.

1. Trier par prix croissant

Du moins cher au plus cher

Pour trier des nombres dans l'ordre croissant, on fait : a - b. 

produits.sort((a, b) => a.prix - b.prix);

console.log(produits);

Résultat attendu :

Souris   25
Lampe    35
Clavier  49
Chaise   89
Écran    199
Bureau   249

2. Trier par prix décroissant

Du plus cher au moins cher

Pour trier des nombres dans l'ordre décroissant, on fait : b - a. 

produits.sort((a, b) => b.prix - a.prix);

console.log(produits);

Résultat attendu :

Bureau   249
Écran    199
Chaise   89
Clavier  49
Lampe    35
Souris   25

3. Trier par stock croissant

Du plus petit stock au plus grand stock

produits.sort((a, b) => a.stock - b.stock);

console.log(produits);

Résultat attendu :

Bureau   3
Écran    5
Chaise   8
Clavier  12
Lampe    15
Souris   30

4. Trier par stock décroissant

Du plus grand stock au plus petit stock

produits.sort((a, b) => b.stock - a.stock);

console.log(produits);

Résultat attendu :

Souris   30
Lampe    15
Clavier  12
Chaise   8
Écran    5
Bureau   3

5. Trier par nom inverse

Ordre alphabétique inversé : Z vers A

Pour trier du texte, on utilise localeCompare(). 

produits.sort((a, b) => b.nom.localeCompare(a.nom));

console.log(produits);

Résultat attendu :

Souris
Lampe
Écran
Clavier
Chaise
Bureau

Bonus : ordre alphabétique A vers Z

produits.sort((a, b) => a.nom.localeCompare(b.nom));

console.log(produits);

Résultat attendu :

Bureau
Chaise
Clavier
Écran
Lampe
Souris

6. Filtrer puis trier

Méthode avec forEach(), if et push()

Ici, on veut récupérer seulement les produits de la catégorie informatique, puis les trier par prix croissant. 

let informatique = [];

produits.forEach(produit => {
  if (produit.categorie === "informatique") {
    informatique.push(produit);
  }
});

informatique.sort((a, b) => a.prix - b.prix);

console.log(informatique);
  • forEach() sert à parcourir le tableau.
  • if sert à vérifier une condition.
  • push() sert à ajouter dans un tableau.
  • sort() sert à trier le résultat.

Même résultat avec filter()

La méthode filter() crée un nouveau tableau avec uniquement les éléments qui respectent une condition. 

const informatique = produits
  .filter(produit => produit.categorie === "informatique")
  .sort((a, b) => a.prix - b.prix);

console.log(informatique);
Ta méthode avec forEach() est très bonne pour apprendre. La méthode avec filter() est simplement plus courte.

7. Trier avec un calcul

Trier par valeur totale du stock

On peut aussi trier un tableau à partir d'un calcul. 

valeurTotale = prix * stock

Tri du plus petit total au plus grand total :

produits.sort((a, b) => {
  return a.prix * a.stock - b.prix * b.stock;
});

console.log(produits);

Version plus lisible :

produits.sort((a, b) => {
  const totalA = a.prix * a.stock;
  const totalB = b.prix * b.stock;

  return totalA - totalB;
});

console.log(produits);

Résultat attendu :

Lampe    35 * 15  = 525
Clavier  49 * 12  = 588
Chaise   89 * 8   = 712
Bureau   249 * 3  = 747
Souris   25 * 30  = 750
Écran    199 * 5  = 995

8. Afficher le produit le plus cher

Trier puis récupérer le premier élément

Pour afficher le produit le plus cher, on trie par prix décroissant, puis on récupère le premier élément avec [0]. 

const produitsTries = [...produits].sort((a, b) => b.prix - a.prix);

console.log(produitsTries[0]);

Résultat attendu :

{ nom: "Bureau", prix: 249, stock: 3, categorie: "bureau" }

9. Ne pas modifier le tableau original

Créer une copie avant de trier

Comme sort() modifie le tableau original, il est souvent préférable de créer une copie avec [...tableau]. 

const produitsTries = [...produits];

produitsTries.sort((a, b) => b.prix - a.prix);

console.log("Tableau original :", produits);
console.log("Tableau trié :", produitsTries);

Version courte :

const produitsTries = [...produits].sort((a, b) => b.prix - a.prix);

console.log(produitsTries);
[...produits] crée une copie du tableau. Ensuite, le tri se fait sur la copie, pas sur l'original.

10. Trier un tableau de dépenses

Tableau de départ

const depenses = [
  { nom: "Courses", montant: 80, date: "2026-04-10" },
  { nom: "Essence", montant: 50, date: "2026-04-08" },
  { nom: "Internet", montant: 30, date: "2026-04-01" },
  { nom: "Restaurant", montant: 45, date: "2026-04-12" }
];

Trier par montant croissant

depenses.sort((a, b) => a.montant - b.montant);

console.log(depenses);

Trier par montant décroissant

depenses.sort((a, b) => b.montant - a.montant);

console.log(depenses);

Trier par date du plus récent au plus ancien

depenses.sort((a, b) => new Date(b.date) - new Date(a.date));

console.log(depenses);

Trier par date du plus ancien au plus récent

depenses.sort((a, b) => new Date(a.date) - new Date(b.date));

console.log(depenses);

Trier par nom alphabétique

depenses.sort((a, b) => a.nom.localeCompare(b.nom));

console.log(depenses);

Résumé à retenir

Objectif Méthode
Nombre croissant a.nombre - b.nombre
Nombre décroissant b.nombre - a.nombre
Texte A vers Z a.nom.localeCompare(b.nom)
Texte Z vers A b.nom.localeCompare(a.nom)
Date ancienne vers récente new Date(a.date) - new Date(b.date)
Date récente vers ancienne new Date(b.date) - new Date(a.date)
Copier avant de trier [...tableau].sort(...)
