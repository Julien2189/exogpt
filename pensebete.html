Définition

Qu’est-ce que le HTML ?

🏗️

Le squelette d’une page

Le HTML sert à organiser le contenu d’une page web : titres, paragraphes, liens, images, listes ou formulaires.

🏷️

Un langage de balisage

HTML signifie HyperText Markup Language. Ce n’est pas un langage de programmation, mais un langage de structure.

🧩

La base du frontend

Le HTML fonctionne avec le CSS pour le design et avec JavaScript pour rendre la page interactive.

Cours 3

Syntaxe générale du HTML

💻

L’environnement de travail

Pour coder une page web, on utilise souvent un IDE, c’est-à-dire un environnement de développement intégré. Il permet d’écrire, tester, organiser et corriger son code plus facilement.

🏷️

Les balises HTML

Le HTML est construit avec des balises entourées par des chevrons. La majorité des balises fonctionnent avec une balise ouvrante et une balise fermante.

🧱

La structure de la page

Un document HTML possède une structure de base avec <!DOCTYPE html>, <html>, <head> et <body>.

01

La ligne <!DOCTYPE html> indique au navigateur que le document utilise HTML5.

02

La balise <html> est l’élément racine : elle contient toute la page HTML.

03

La balise <head> contient les informations invisibles de la page : titre, métadonnées, lien CSS, encodage, SEO.

04

La balise <body> contient tout ce qui est visible par l’utilisateur : titres, textes, images, boutons, liens et formulaires.

05

Une balise HTML peut avoir des attributs. Par exemple, lang="fr" indique que la page est en français.

06

La balise <div> sert à créer des blocs ou des conteneurs pour organiser les éléments d’une page.

07

Les balises <header>, <nav>, <section>, <article>, <aside> et <footer> donnent du sens à la structure de la page.

08

Utiliser des balises sémantiques rend le code plus clair, plus facile à maintenir et meilleur pour le référencement naturel.

Exemple

Structure HTML complète

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ma page web</title>
    <link rel="stylesheet" href="style.css">
  </head>

  <body>
    <header>
      <nav>
        <a href="#">Accueil</a>
      </nav>
    </header>

    <main>
      <section>
        <h1>Titre principal</h1>
        <p>Mon premier paragraphe.</p>
      </section>
    </main>

    <footer>
      <p>Pied de page</p>
    </footer>
  </body>
</html>

Cours 3

L’organisation du texte en HTML

🧱

Structurer le texte

Le HTML permet d’organiser le contenu textuel d’une page web. Une bonne structure aide l’utilisateur à lire plus facilement et améliore aussi le référencement naturel du site.

🔠

Les titres HTML

Les balises <h1> à <h6> servent à créer une hiérarchie de titres. <h1> représente le titre principal, puis <h2>, <h3>, etc. servent aux sous-titres.

📄

Les paragraphes

La balise <p> permet de créer un paragraphe. Pour garder une bonne lisibilité, il faut éviter les longs blocs de texte. La règle simple à retenir : une idée = un paragraphe.

01

La balise <title> se place dans le <head> et affiche le titre dans l’onglet du navigateur.

02

La balise <h1> se place dans le <body> et affiche le titre principal visible sur la page.

03

Une liste non ordonnée se crée avec <ul>. Elle sert pour des éléments sans ordre précis.

04

Une liste ordonnée se crée avec <ol>. Elle sert pour des étapes ou des éléments numérotés.

05

Chaque élément d’une liste doit être placé dans une balise <li>.

06

Les balises block créent un bloc sur la page : <div>, <p>, <h1>, <ul>, <ol>.

07

Les balises inline restent dans la ligne du texte : <a>, <span>, <img>, <button>.

08

La balise <div> sert à créer un conteneur ou une zone dans la page.

09

La balise <span> sert à appliquer un style sur une petite partie d’un texte sans créer un nouveau bloc.

10

La balise <br> ajoute un saut de ligne, tandis que <hr> ajoute une ligne horizontale de séparation.

Mise en forme

Les balises importantes pour le texte

💪

Gras et importance

<b> met le texte en gras uniquement pour l’apparence. <strong> met aussi le texte en gras, mais indique en plus que le contenu est important.

✍️

Italique et emphase

<i> met le texte en italique pour l’apparence. <em> indique une emphase, donc une importance particulière dans le sens du texte.

🖍️

Surlignage et petits textes

<mark> permet de surligner une partie du texte. <small> permet d’afficher un texte secondaire plus petit, comme une note ou une précision.

Exemples pratiques

<p>
  Voici un paragraphe court et clair.
</p>

<h2>Sous-titre</h2>

<ul>
  <li>Un élément de liste</li>
  <li>Un autre élément</li>
</ul>

<ol>
  <li>Première étape</li>
  <li>Deuxième étape</li>
</ol>

Style en HTML

Attribut style, balise style et class

🎨

L’attribut style

L’attribut style permet d’ajouter du CSS directement sur une balise HTML. C’est pratique pour tester, mais moins propre pour un vrai projet.

🧾

La balise style

La balise <style> permet d’écrire du CSS dans le document HTML, souvent dans la partie <head>.

🏷️

L’attribut class

L’attribut class permet de donner une étiquette à plusieurs éléments pour leur appliquer le même style CSS. 

<style>
  .titre-vert {
    color: green;
    text-align: center;
  }
</style>

<h2 class="titre-vert">Mon sous-titre</h2>

<p style="color: blue; font-size: 18px;">
  Ceci est un paragraphe bleu.
</p>

Cours 4

Un langage hypertexte

🔗

Qu’est-ce qu’un lien hypertexte ?

Un lien hypertexte est un élément cliquable qui permet de naviguer entre plusieurs contenus : une autre page, une section, un fichier, une adresse e-mail ou un site externe.

🏷️

La balise <a>

En HTML, les liens sont créés avec la balise <a>. Le contenu placé entre <a> et </a> devient cliquable.

🎯

L’attribut href

L’attribut href indique la destination du lien. Il peut pointer vers une page, une section, un fichier, une adresse mail ou un site externe.

01

Les liens hypertextes servent à améliorer la navigation dans un site web.

02

Un lien peut être placé sur du texte, une image ou même un bouton.

03

Un lien interne permet de relier plusieurs pages d’un même site ou plusieurs sections d’une même page.

04

Un lien externe redirige l’utilisateur vers un autre site web.

05

Un lien courriel utilise mailto: pour permettre à l’utilisateur d’envoyer un e-mail.

06

Un lien vers un fichier peut permettre d’ouvrir ou de télécharger un document comme un PDF, une image ou une vidéo.

07

L’attribut target="_blank" permet d’ouvrir un lien dans un nouvel onglet.

08

L’attribut download permet de proposer directement le téléchargement d’un fichier.

09

L’attribut title affiche une information supplémentaire au survol du lien.

10

La balise <link> ne sert pas à créer un lien cliquable : elle sert à relier le document HTML à une ressource externe, comme un fichier CSS.

Exemples pratiques

Créer des liens en HTML

<!-- Lien vers un site externe -->
<a href="https://developer.mozilla.org" target="_blank">
  Documentation MDN
</a>

<!-- Lien vers une autre page du site -->
<a href="contact.html">Page contact</a>

<!-- Lien vers une section de la même page -->
<a href="#projets">Voir mes projets</a>

<section id="projets">
  <h2>Mes projets</h2>
</section>

<!-- Lien pour envoyer un e-mail -->
<a href="mailto:contact@example.com">Me contacter</a>

<!-- Lien pour télécharger un fichier -->
<a href="./assets/cv.pdf" download>Télécharger mon CV</a>

Cours 5

Le multimédia en HTML

🖼️

Les fichiers multimédia

Un fichier multimédia peut contenir différents types de médias : image, vidéo, audio, graphique ou carte interactive. Il permet de rendre une page web plus visuelle, attractive et immersive.

🌄

Les images en HTML

Pour afficher une image, on utilise la balise <img>. Elle fonctionne avec l’attribut src pour indiquer le chemin de l’image et l’attribut alt pour ajouter un texte alternatif.

🎬

Audio et vidéo

Les balises <audio> et <video> permettent d’intégrer des sons et des vidéos dans une page web. L’attribut controls affiche les boutons de lecture.

01

Le multimédia améliore l’expérience utilisateur en rendant le contenu plus vivant, plus clair et plus agréable à consulter.

02

La balise <img> est une balise autofermante : elle n’a pas besoin de balise de fermeture.

03

L’attribut src indique l’URL ou le chemin du fichier à afficher dans la page.

04

L’attribut alt décrit l’image. Il est important pour l’accessibilité, le référencement et le cas où l’image ne se charge pas.

05

La balise <picture> permet d’afficher des images différentes selon la taille de l’écran grâce aux balises <source>.

06

Les formats d’image courants sont JPG, PNG, GIF, SVG, WebP et AVIF.

07

Les formats WebP et AVIF sont souvent intéressants pour optimiser le chargement des images.

08

L’attribut autoplay doit être utilisé avec prudence, car il peut lancer un son ou une vidéo sans action de l’utilisateur.

09

La balise <iframe> permet d’intégrer du contenu externe comme une vidéo YouTube, une vidéo Vimeo ou une carte interactive.

10

Les balises <embed> et <object> permettent aussi d’intégrer du contenu externe, comme un PDF, un SVG ou certains fichiers multimédia.

Exemples pratiques

Ajouter du multimédia en HTML

<!-- Ajouter une image -->
<img src="./assets/image.jpg" alt="Description de l'image">

<!-- Image responsive avec picture -->
<picture>
  <source media="(min-width: 1200px)" srcset="image-desktop.jpg">
  <source media="(min-width: 768px)" srcset="image-tablette.jpg">
  <img src="image-mobile.jpg" alt="Image responsive">
</picture>

<!-- Ajouter une vidéo -->
<video controls>
  <source src="./assets/video.webm" type="video/webm">
  <source src="./assets/video.mp4" type="video/mp4">
</video>

<!-- Ajouter un fichier audio -->
<audio controls>
  <source src="./assets/audio.mp3" type="audio/mpeg">
  <source src="./assets/audio.ogg" type="audio/ogg">
</audio>

<!-- Intégrer une vidéo externe -->
<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/id-de-la-video">
</iframe>

Cours 6

Introduction au CSS

🎨

Qu’est-ce que le CSS ?

CSS signifie Cascading Style Sheets, c’est-à-dire feuilles de style en cascade. Il permet de modifier l’apparence d’une page HTML : couleurs, polices, marges, tailles, bordures et mise en page.

🧱

HTML + CSS

Le HTML sert à structurer le contenu, tandis que le CSS sert à le présenter visuellement. Grâce au CSS, on peut rendre une page web plus agréable, plus lisible et plus responsive.

📁

Feuille de style externe

La méthode recommandée consiste à placer le CSS dans un fichier externe, souvent nommé style.css, puis à le relier au fichier HTML avec la balise <link>.

01

Le CSS permet de séparer le contenu HTML de la présentation visuelle.

02

Une règle CSS est composée d’un sélecteur, d’une propriété et d’une valeur.

03

La syntaxe générale est : sélecteur { propriété: valeur; }.

04

Le caractère : sépare la propriété de sa valeur, et le ; sépare les déclarations CSS.

05

Le CSS peut être écrit dans un fichier externe, dans une balise <style> ou directement dans une balise HTML avec l’attribut style.

06

La feuille de style externe est la méthode la plus propre pour un vrai projet, car elle facilite la maintenance du site.

07

Une classe CSS commence par un point : .ma-classe. Elle peut être utilisée sur plusieurs éléments.

08

Un ID CSS commence par un dièse : #mon-id. Il doit être unique dans une page HTML.

09

La cascade signifie que si deux règles CSS ciblent le même élément, la dernière règle ou la règle la plus spécifique peut prendre le dessus.

10

Les pseudo-classes comme :hover, :focus, :visited ou :active permettent de modifier le style selon l’état d’un élément.

Exemples pratiques

Écrire du CSS

<!-- Dans le fichier HTML -->
<link rel="stylesheet" href="style.css">


/* Dans le fichier style.css */

body {
  font-family: Arial, sans-serif;
  background-color: #f5f5f5;
  color: #222;
}

h1 {
  color: tomato;
  text-align: center;
  font-size: 36px;
}

p {
  font-size: 18px;
  line-height: 1.6;
}

.ma-classe {
  color: blue;
  font-weight: bold;
}

#mon-id {
  background-color: yellow;
  padding: 20px;
}

a {
  color: darkgreen;
}

a:hover {
  color: tomato;
  text-decoration: none;
}

Cours 7

CSS3 : Le stylage du texte

🔤

Mettre en forme le texte

Le CSS permet de contrôler l’apparence du texte : couleur, police, taille, graisse, italique, alignement, espacement, interligne et décoration.

🎨

Les couleurs

Les couleurs peuvent être écrites avec des mots-clés comme red, en hexadécimal comme #FF0000, ou avec rgb() et rgba().

✍️

Les polices

Les propriétés font-family, font-size, font-weight et font-style permettent de personnaliser la typographie d’une page.

01

La propriété color permet de modifier la couleur du texte.

02

La propriété background-color permet de modifier la couleur de fond d’un élément.

03

font-family définit la police utilisée. Il est conseillé d’ajouter une police générique à la fin, comme sans-serif, serif ou monospace.

04

font-size définit la taille du texte. On peut utiliser des unités comme px, %, em ou rem.

05

font-weight définit l’épaisseur du texte. Par exemple : normal, bold, ou des valeurs de 100 à 900.

06

font-style permet notamment de mettre un texte en italique avec la valeur italic.

07

text-align permet d’aligner le texte : left, center, right ou justify.

08

text-decoration permet de souligner, surligner ou barrer un texte avec underline, overline ou line-through.

09

text-transform permet de transformer le texte en majuscules, minuscules ou avec une majuscule au début de chaque mot.

10

line-height, letter-spacing et word-spacing permettent de gérer l’espace entre les lignes, les lettres et les mots.

Exemples pratiques

Styliser du texte en CSS

/* Couleur du texte et couleur de fond */
.titre {
  color: tomato;
  background-color: #f5f5f5;
}

/* Police de caractères */
.paragraphe {
  font-family: Arial, sans-serif;
  font-size: 1rem;
  font-weight: normal;
  font-style: normal;
}

/* Texte en gras */
.texte-gras {
  font-weight: bold;
}

/* Texte en italique */
.texte-italique {
  font-style: italic;
}

/* Alignement du texte */
.texte-centre {
  text-align: center;
}

.texte-justifie {
  text-align: justify;
}

/* Décoration du texte */
.texte-souligne {
  text-decoration: underline;
}

.texte-barre {
  text-decoration: line-through;
}

/* Transformation du texte */
.texte-majuscule {
  text-transform: uppercase;
}

.texte-capitalise {
  text-transform: capitalize;
}

/* Espacement du texte */
.texte-espace {
  line-height: 1.6;
  letter-spacing: 1px;
  word-spacing: 4px;
}

Polices et unités

Aller plus loin avec le texte

📏

L’unité rem

L’unité rem se base sur la taille de police de l’élément racine. Par défaut, 1rem vaut souvent 16px dans le navigateur.

🌈

Opacité des couleurs

Avec rgba(), on peut ajouter une transparence. Exemple : rgba(255, 0, 0, 0.25) donne un rouge avec 25 % d’opacité.

🖋️

Polices externes

On peut utiliser des polices externes avec Google Fonts ou charger une police personnelle avec la règle CSS @font-face.

Exemple

Utiliser une police Google Fonts

<!-- Dans le fichier HTML, dans le head -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Dancing+Script&display=swap" rel="stylesheet">


/* Dans le fichier CSS */

h1,
h2 {
  font-family: "Dancing Script", cursive;
}

Cours 8

Structure logique HTML5 et modèle de boîtes CSS3

🧱

Structure logique HTML5

HTML5 propose des balises sémantiques qui donnent du sens à la page : <header>, <nav>, <main>, <section>, <article>, <aside> et <footer>.

📦

Le modèle de boîtes

En CSS, chaque élément HTML peut être vu comme une boîte composée du contenu, du padding, de la bordure et de la marge extérieure.

Accessibilité et clarté

Propriétés importantes du Box Model

01

content représente le contenu réel de l’élément : texte, image, bouton, etc.

02

padding ajoute un espace entre le contenu et la bordure.

03

border ajoute une bordure autour du contenu et du padding.

04

Cours 9

Les sélecteurs CSS

🎯

À quoi servent les sélecteurs ?

Les sélecteurs CSS permettent de cibler précisément des éléments HTML afin de leur appliquer un style : couleur, taille, marge, bordure, arrière-plan ou effet au survol.

🏷️

Les sélecteurs courants

Les plus utilisés sont le sélecteur d’élément, le sélecteur de classe et le sélecteur d’ID. Ils permettent de styliser une balise, plusieurs éléments ou un élément unique.

🧠

Les sélecteurs avancés

CSS permet aussi de cibler des éléments selon leur relation, leurs attributs, leur état ou certaines parties précises d’un élément grâce aux pseudo-classes et pseudo-éléments.

01

Une règle CSS est composée d’un sélecteur, d’une ou plusieurs propriétés et de leurs valeurs.

02

Le sélecteur d’élément cible toutes les balises du même type. Exemple : p cible tous les paragraphes.

03

Le sélecteur de classe commence par un point : .important. Une classe peut être utilisée sur plusieurs éléments.

04

Le sélecteur d’ID commence par un dièse : #introduction. Un ID doit être unique dans une page HTML.

05

Le sélecteur descendant cible un élément situé à l’intérieur d’un autre. Exemple : ul p cible les paragraphes contenus dans une liste.

06

Le sélecteur enfant direct utilise >. Exemple : ul > li cible uniquement les li directement enfants du ul.

07

Le sélecteur d’adjacence immédiate utilise +. Exemple : h2 + p cible le paragraphe placé juste après un h2.

08

Le sélecteur général d’adjacence utilise ~. Exemple : h2 ~ p cible les paragraphes frères placés après un h2.

09

Les sélecteurs d’attributs permettent de cibler des éléments selon leurs attributs, par exemple [title] ou [href^="https"].

10

Les pseudo-classes comme :hover, :focus ou :first-child permettent de styliser un élément selon son état ou sa position.

Exemples pratiques

Les sélecteurs CSS courants

/* Sélecteur d’élément : cible tous les paragraphes */
p {
  color: blue;
  font-size: 16px;
}

/* Sélecteur de classe : cible tous les éléments avec class="important" */
.important {
  font-weight: bold;
  font-size: 18px;
}

/* Sélecteur d’ID : cible l’élément avec id="introduction" */
#introduction {
  font-size: 20px;
  font-style: italic;
}

/* Sélecteur descendant : cible les p dans les ul */
ul p {
  color: green;
}

/* Sélecteur enfant direct : cible uniquement les li directement dans ul */
ul > li {
  font-weight: bold;
}

/* Sélecteur d’adjacence immédiate : cible le p juste après un h2 */
h2 + p {
  margin-top: 10px;
}

/* Sélecteur général d’adjacence : cible les p placés après un h2 */
h2 ~ p {
  color: darkcyan;
}

Sélecteurs avancés

Attributs, pseudo-classes et pseudo-éléments

🔎

Sélecteurs d’attributs

Ils permettent de cibler des éléments selon la présence ou la valeur d’un attribut HTML, par exemple un lien HTTPS, un fichier PDF ou un élément avec un attribut title.

🖱️

Pseudo-classes

Elles ciblent un élément selon son état ou sa position : survol de souris, focus clavier, premier enfant, dernier enfant ou élément exclu avec :not().

✨

Pseudo-éléments

Ils permettent de styliser une partie d’un élément ou d’ajouter du contenu virtuel avec ::before, ::after, ::first-letter ou ::first-line.

Exemples avancés

Sélecteurs CSS avancés

/* Cible tous les éléments qui possèdent un attribut title */
[title] {
  border-bottom: 1px dotted black;
}

/* Cible les liens qui commencent par https */
[href^="https"] {
  color: green;
}

/* Cible les liens qui se terminent par .pdf */
[href$=".pdf"] {
  color: red;
}

/* Cible les classes qui contiennent le mot button */
[class*="button"] {
  padding: 10px 20px;
}

/* Survol de la souris */
button:hover {
  background-color: blue;
  color: white;
}

/* Élément qui reçoit le focus */
input:focus {
  border: 2px solid tomato;
}

/* Premier élément d’une liste */
li:first-child {
  font-weight: bold;
}

/* Dernier élément d’une liste */
li:last-child {
  color: darkcyan;
}

/* Cible tous les éléments sauf les paragraphes */
:not(p) {
  box-sizing: border-box;
}

/* Ajoute du contenu avant un titre */
h1::before {
  content: "Titre : ";
  font-weight: bold;
}

/* Ajoute du contenu après un paragraphe */
p::after {
  content: " ✔";
}

/* Style sur la première lettre d’un paragraphe */
p::first-letter {
  font-size: 2em;
  font-weight: bold;
}

Flexbox

Comprendre le conteneur flex

↔️

Axe principal

Par défaut, Flexbox place les éléments sur une ligne horizontale. Cette direction correspond à flex-direction: row.

↕️

Axe transversal

L’axe transversal est perpendiculaire à l’axe principal. Si les éléments sont en ligne, l’axe transversal est vertical.

🔁

Retour à la ligne

Avec flex-wrap: wrap, les éléments peuvent passer à la ligne si l’espace disponible devient insuffisant.

01

Pour activer Flexbox, on utilise display: flex sur l’élément parent.

02

Les enfants directs du conteneur deviennent automatiquement des éléments flex.

03

flex-direction définit la direction des éléments : row, row-reverse, column ou column-reverse.

04

flex-wrap définit si les éléments restent sur une ligne ou peuvent passer à la ligne.

05

justify-content aligne les éléments sur l’axe principal.

06

align-items aligne les éléments sur l’axe transversal.

07

space-between répartit les éléments avec de l’espace entre eux.

08

center permet de centrer les éléments facilement.

La création de formulaires HTML

📝

À quoi sert un formulaire ?

Un formulaire permet à l’utilisateur d’envoyer des informations : inscription, connexion, contact, commentaire, choix dans une liste ou envoi de fichier.

📦

La balise <form>

La balise <form> regroupe tous les champs du formulaire. Elle utilise les attributs action et method pour définir où et comment envoyer les données.

♿

Accessibilité

Les balises <label> sont importantes : elles indiquent clairement à l’utilisateur quelle information est attendue dans chaque champ.

01

La balise <form> permet de créer un formulaire HTML.

02

L’attribut action indique la page ou l’URL qui recevra les données du formulaire.

03

L’attribut method indique la méthode d’envoi : GET ou POST.

04

La méthode GET transmet les données dans l’URL. Elle est plutôt adaptée aux recherches ou aux filtres.

05

La méthode POST transmet les données dans une requête HTTP. Elle est préférable pour les données sensibles comme un mot de passe.

06

La balise <input> permet de créer un champ de saisie. C’est une balise orpheline : elle n’a pas de balise fermante.

07

L’attribut type indique le type de champ : text, email, password, number, radio, checkbox ou submit.

08

Pour associer un label à un champ, on utilise for sur le label et id sur le champ avec la même valeur.

09

La balise <textarea> permet de créer une zone de texte longue, utile pour un commentaire ou un message.

10

Les balises <fieldset> et <legend> permettent de regrouper et nommer une partie du formulaire.

La création de tableaux HTML

📊

À quoi sert un tableau ?

Un tableau HTML sert à organiser des données sous forme de lignes et de colonnes. Il est utile pour présenter des informations structurées de manière claire et lisible.

🧱

Structure de base

Un tableau est créé avec la balise <table>. Les lignes sont créées avec <tr>, les cellules avec <td> et les en-têtes avec <th>.

♿

Accessibilité

Un tableau doit être bien structuré pour être compréhensible par tous, y compris avec un lecteur d’écran. Les balises <thead>, <tbody>, <tfoot> et l’attribut scope améliorent cette structure.

01

La balise <table> contient l’ensemble du tableau.

02

La balise <tr> signifie Table Row et permet de créer une ligne.

03

La balise <td> signifie Table Data et permet de créer une cellule contenant une donnée.

04

La balise <th> signifie Table Header et permet de créer une cellule d’en-tête.

05

Un tableau doit avoir une structure cohérente : chaque ligne doit contenir le bon nombre de cellules, sauf si on utilise rowspan ou colspan.

06

Les anciens attributs HTML comme border, bgcolor, cellpadding ou cellspacing sont obsolètes : on utilise le CSS pour styliser un tableau.

07

La balise <caption> ajoute une légende au tableau. Elle doit être placée juste après l’ouverture de <table>.

08

colspan permet de fusionner plusieurs colonnes.

09

rowspan permet de fusionner plusieurs lignes.

10

Les tableaux servent à afficher des données, pas à construire la mise en page d’un site web.

Le layout avec CSS Grid

▦

Qu’est-ce que CSS Grid ?

CSS Grid est un module CSS qui permet de créer des mises en page sous forme de grille, avec des lignes et des colonnes. Il est très utile pour organiser précisément les éléments d’une page web.

📐

Lignes et colonnes

Une grille CSS est composée de colonnes et de lignes. On peut définir leur taille avec des pixels, des pourcentages ou l’unité flexible fr.

📱

Mise en page responsive

Avec CSS Grid et les media queries, une page peut s’adapter aux écrans larges, tablettes et mobiles en modifiant automatiquement le nombre de colonnes.

01

Pour créer une grille, on applique display: grid sur un conteneur parent.

02

Les éléments enfants directs du conteneur deviennent les éléments de la grille.

03

grid-template-columns permet de définir les colonnes de la grille.

04

grid-template-rows permet de définir les lignes de la grille.

05

L’unité fr signifie fraction. Elle permet de partager l’espace disponible entre les colonnes ou les lignes.

06

La fonction repeat() permet de répéter une valeur. Exemple : repeat(3, 1fr) crée trois colonnes égales.

07

La propriété gap permet d’ajouter un espace entre les éléments de la grille.

08

grid-column permet de placer un élément sur une ou plusieurs colonnes.

09

grid-row permet de placer un élément sur une ou plusieurs lignes.

10

Les media queries permettent de changer la grille selon la taille de l’écran.

Exemple simple

Créer une grille à deux colonnes

  gap: 20px;
}

  border-radius: 10px;
  text-align: center;

🧩

Le responsive design

📱

Qu’est-ce que le responsive design ?

Le responsive design est une méthode de conception qui permet à un site web de s’adapter automatiquement à la taille de l’écran : ordinateur, tablette ou smartphone.

🧩

Une meilleure expérience utilisateur

Un site responsive reste lisible, clair et agréable à utiliser sur tous les appareils. Il améliore la navigation et évite les débordements ou les textes trop petits.

🔎

Un avantage pour le SEO

Un site adapté aux mobiles peut améliorer le référencement naturel, car les moteurs de recherche privilégient les sites offrant une bonne expérience utilisateur.

01

Le responsive design permet à une page web de s’adapter aux différentes tailles d’écran.

02

La technique du responsive design a été popularisée en 2010 par Ethan Marcotte.

03

Un site responsive évite d’avoir une version différente du site pour chaque appareil.

04

Le fluid design utilise des unités flexibles comme les pourcentages au lieu de tailles fixes en pixels.

05

Le mobile-first consiste à concevoir d’abord la version mobile, puis à adapter le site aux écrans plus grands.

06

Les images doivent être flexibles pour éviter de dépasser de leur conteneur.

07

Les media queries permettent d’appliquer des styles CSS différents selon la taille de l’écran.

08

Il faut tester son site avec l’inspecteur du navigateur pour vérifier l’affichage sur mobile, tablette et ordinateur.

09

Il faut éviter les pop-ups trop envahissantes et les publicités gênantes sur mobile.

10

Il faut réduire le poids des ressources : images, scripts et fichiers CSS, pour améliorer le temps de chargement.

Images responsives

Adapter les images à l’écran

img {
  max-width: 100%;
  height: auto;
}

max-width: 100% empêche l’image de dépasser son conteneur. height: auto conserve les proportions de l’image.

Adapter le CSS selon la taille de l’écran

🖥️

Écran large

Sur ordinateur, on peut afficher plusieurs colonnes, des menus horizontaux et davantage d’espace entre les éléments.

📲

Tablette

Sur tablette, on réduit souvent le nombre de colonnes et on adapte les espacements pour garder une bonne lisibilité.

📱

Mobile

Sur mobile, on privilégie souvent une seule colonne, des boutons plus grands et une navigation simplifiée.

    flex-direction: column;

    width: 100%;
  }
}

Comprendre le préprocesseur Sass

💎

Qu’est-ce que Sass ?

Sass est un préprocesseur CSS. Il permet d’écrire du code plus organisé, puis de le compiler en CSS classique lisible par le navigateur.

🔁

Réutiliser le code

Sass permet d’utiliser des variables, des mixins, des fonctions et des boucles pour éviter de répéter le même code.

🧹

Grâce à Sass, le CSS devient plus facile à maintenir, surtout dans les projets plus grands.

01

Sass signifie Syntactically Awesome Style Sheets.

02

03

Une variable Sass commence par le symbole $.

04

Les mixins permettent de créer des blocs de styles réutilisables avec @mixin et @include.

Variables Sass

Définir une valeur réutilisable

$primary-color: #007bff;
$font-family: Arial, sans-serif;

body {
  font-family: $font-family;
  color: #222;
}

.button {
  background-color: $primary-color;
  color: white;
}

Ici, la couleur principale est stockée dans une variable. Si tu veux changer la couleur du site, tu modifies seulement $primary-color.

Mixins Sass

Créer un bloc de code réutilisable

@mixin button($bg-color) {
  background-color: $bg-color;
  color: white;
  padding: 10px 20px;
  border-radius: 5px;
  text-transform: uppercase;
  font-weight: bold;
}

.button-primary {
  @include button(#0074d9);
}

.button-secondary {
  @include button(#7fdbff);
}

Le mixin button évite de répéter les mêmes propriétés pour chaque bouton.

Fonctions Sass

Créer une fonction personnalisée

@function px-to-rem($px-value) {
  $rem-value: $px-value / 16;
  @return #{$rem-value}rem;
}

h1 {
  font-size: px-to-rem(32px);
}

Cette fonction convertit une valeur en pixels vers une valeur en rem. Par exemple, 32px devient 2rem.

Boucles Sass

Générer plusieurs classes automatiquement

@for $i from 1 through 5 {
  .box-#{$i} {
    width: 20px * $i;
    height: 20px * $i;
  }
}

$colors: red, green, blue;

@each $color in $colors {
  .text-#{$color} {
    color: $color;
  }
}

Les boucles peuvent être très utiles, mais il faut les utiliser avec prudence pour éviter de générer trop de CSS inutile.

Compilation Sass

Transformer SCSS en CSS

sass styles.scss styles.css

Cette commande compile le fichier styles.scss en fichier styles.css, utilisable dans une page HTML.

Le référencement des sites Web

🔎

Qu’est-ce que le référencement ?

Le référencement Web désigne l’ensemble des techniques qui permettent à un site d’être trouvé dans les moteurs de recherche comme Google. L’objectif est d’améliorer sa visibilité.

📈

Pourquoi c’est important ?

Un site bien référencé attire plus de visiteurs. La majorité des internautes ne dépasse pas la première page des résultats de recherche.

🧠

SEO et SEA

Le SEO correspond au référencement naturel gratuit. Le SEA correspond au référencement payant avec des annonces sponsorisées.

01

Le référencement permet à un site d’être présent dans les moteurs de recherche.

02

Être référencé ne suffit pas : il faut aussi être bien positionné dans les résultats de recherche.

03

La SERP signifie Search Engine Results Page. C’est la page de résultats affichée après une recherche.

04

Le but du SEO est d’apparaître le plus haut possible dans les résultats naturels des moteurs de recherche.

05

Le résultat numéro 1 sur Google reçoit beaucoup plus de clics que les résultats suivants. Le document indique environ 28,5 % des clics pour la première position.

06

Le SEO repose sur le contenu, les mots clés, les balises HTML, la vitesse du site, les liens et la qualité globale de la page.

07

Le SEA permet d’acheter des mots clés pour apparaître dans les annonces payantes.

08

Les moteurs de recherche favorisent les contenus longs, uniques, originaux et bien structurés.

09

Les mots clés doivent être placés naturellement dans les titres, textes, URL, images, liens, métatitres et métadescriptions.

10

Il faut éviter de répéter excessivement les mots clés, car cela peut être considéré comme une mauvaise pratique par les moteurs de recherche.

SEO

Les balises importantes pour le référencement

🏷️

Balise title

La balise <title> donne le titre de la page. Elle apparaît dans l’onglet du navigateur et dans les résultats de recherche. Elle ne doit pas dépasser environ 60 caractères.

📝

Métadescription

La balise meta description résume le contenu de la page dans la SERP. Elle doit être claire, attractive et contenir environ 150 à 160 caractères.

🖼️

Attribut alt

L’attribut alt décrit une image. Il aide les moteurs de recherche à comprendre l’image et améliore aussi l’accessibilité.

Comprendre DNS, domaine et TLD

🌐

Adresse IP

Une adresse IP permet d’identifier un serveur sur Internet. Exemple : 172.217.17.68.

🏷️

Domaine

Le domaine correspond au nom principal avant l’extension. Dans google.com, le domaine est google.

🔚

TLD

Le TLD est l’extension du nom de domaine. Exemples : .fr, .com, .org. 

Nom de domaine : service-public.fr

Domaine : service-public
TLD : fr

Hébergement

Les principales solutions d’hébergement

🏠

Hébergement mutualisé

Plusieurs sites partagent le même serveur. C’est une solution simple et peu coûteuse, adaptée aux petits sites vitrines ou aux projets avec peu de trafic.

🖥️

Serveur dédié physique

Le serveur est réservé à un seul client. Il offre plus de contrôle et de puissance, mais demande plus de compétences et coûte plus cher.

☁️

Serveur dédié virtuel

Le VPS est une machine virtuelle créée sur un serveur physique. Il offre un bon compromis entre coût, contrôle et performances.

⚙️

Cloud / PaaS

Le cloud permet d’utiliser des ressources à la demande. Les plateformes PaaS simplifient le déploiement sans gérer toute l’infrastructure serveur.

🚀

Netlify

Netlify est pratique pour déployer des sites front-end : HTML, CSS, JavaScript, React, Vue ou portfolios.

🧩

Heroku

Heroku est une plateforme PaaS plus complète, utile pour héberger des applications avec back-end comme Node.js, Python, PHP ou Symfony.

FTP

Transférer les fichiers avec FTP

📁

FTP

FTP signifie File Transfert Protocol. Il permet de transférer des fichiers entre un ordinateur local et un serveur distant.

🖧

Serveur FTP

Le serveur FTP est un logiciel installé sur la machine distante. Il donne accès à une arborescence de fichiers.

💻

Client FTP

Le client FTP est le logiciel utilisé sur ton ordinateur pour te connecter au serveur et envoyer les fichiers. Exemple : FileZilla.

01

Le port FTP par défaut pour les commandes est 21.

02

Le port FTP utilisé pour les fichiers est généralement 20.

03

Le protocole FTPS ajoute une couche de sécurité grâce à TLS.

04

Pour se connecter en FTP, il faut connaître l’hôte, le port, l’identifiant et le mot de passe.
